• Q：拍下后的产品什么时候发货？

    A：

    在拼趣多参团成功后除特殊商品外，商家将在48小时内发货。以下情况除外：

    预售及其他特殊情况商品，商品会在约定发货时间内安排发货，发货时间以商品描述页面为准。

  • Q：什么情况下订单会自动取消？

    A：

    1、开团后24小时内未成团的会自动取消订单并退款。

    2、在2分钟内未付款的订单会自动取消。

  • Q：没有收到货/商家发错了地址怎么办？

    A：

    买家可以在已发货订单查看物流信息，并点击查看发货信息、物流单号，物流信息不准确可以联系商家客服核实并要求处理，若商家拒绝处理，可以要求拼趣多客服介入处理。

  • Q：收到产品有问题怎么办？

    A：

    买家可以先联系卖家客服，并提供实物图片确认问题是否属实。如果确认商品问题，或无法说明商品是否合格，卖家应与买家协商解决；若因商品问题商家拒绝处理，可以要求拼趣多客服介入处理。

  • Q：为什么我的优惠券不能使用？

    A：

    请查看一下您的支付金额是否达到了优惠券使用的条件，若已经满足优惠券的使用条件还不能使用可以与商家联系。

  • Q：收到的产品能退换货么？

    A：

    拼趣多商城支持7天无理由退换货，在不影响第二次销售的情况下，可以要求商家无理由退换货。因为商品的特殊性，7天无理由退换货不支持生鲜、虚拟类目。

  • Q：退换货流程

    A：

    1、请务必先与商家联系，说明退换货原因。

    2、若商品符合退换货条件，与商家协商好之后商家客服会提供退货地址和退货流程，按流程操作即可。（买家未联系商家就自行退货导致商家无法收到退回的商品，商家有权不予处理。）

    3、若商品符合退换货条件，商家若不予处理，买家可以联系拼趣多客服介入。

  • Q：退换货规则

    A：

    1、买家须在签收商品之日起7天内向商家提出退换货（按照物流签收后的第二天零时起计算时间，满168小时为7天）。

    2、买家申请退款金额不得高于商品实际付款金额。

    3、若买家退还原因为“7天无理由退换货”，则买家须承担退换货运费。

    4、若商品质量问题、发错、丢件等形式导致退换货，运费全部由商家承担。

    5、若因个人原因对水产肉类、新鲜果蔬、熟食等食品进行拒收，商家有权拒绝对买家进行退款。

    6、若买家收到商品后超过7天再联系商家或拼趣多客服处理退换货需求，商家有权不予处理。

    7、若买家自收到商品之日起7天内根据商家或系统提供的退换货地址进行退换货，商家收到所退商品之日起3天内未有异议但并未处理的，拼趣多官方有权介入处理。（按照物流签收后的第二天零时起计算时间，满72小时为3天）

    8、关于拒收：若买家无任何理由拒绝签收商品，运费由买家承担；以下拒收情况但不仅限于以下几种为商家承担运费：a、在商家未发货前，买家已经申请退款表示取消订单后未经买家统一而商家强行发货导致的拒收。b、买家验货时发现商品破损或者与商品不符导致的拒收。

  • Q：什么是免单拼？

    A：

    免单拼是一种拼团成功后系统会随机抽取参团人员进行免单的拼团形式，获得免单者会直接退款；免单拼由团长发起拼团，参加的人数和免单人数也由团长自定义。

  • Q：参团码怎么使用？

    A：

    复制参团码后直接在首页点击【参团】然后输入参团码就可以跳转相应的商品团。

  • Q：什么是为我点赞？

    A：

    团长开团后邀请相应数量的好友进行下载拼趣多app并且支付0.01元，团满后团长可以免费获得该商品。

  • Q：可以延长收货多少天？

    A：

    点击延长收货可以延长5天，每笔订单只能延长收货一次。

  • Q：关于7天无理由退换货

    A：

    1、7天是指按照物流签收后的第二天零时起计算时间，满168小时为7天

    2、在不影响第二次销售的情况下，可以要求商家无理由退换货。

    3、因为某些商品的特殊性，7天无理由退换货不支持生鲜、虚拟类目。
